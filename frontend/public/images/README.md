# Images Directory

## Hero Image
**File:** `hero-godavari.jpg`
**Description:** The main hero background image showing the Godavari river at sunset with the iconic bridge and traditional boats.

### To Add Your Image:
1. Save your Godavari river sunset image as `hero-godavari.jpg` in this directory
2. Recommended dimensions: 1920x1080 or higher
3. The image will automatically appear as the hero section background

2. Recommended dimensions: 800x800 (square format)
3. Images will automatically appear in the portfolio grid with filtering
